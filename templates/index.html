Schola

This website serves as an example minimalist course directory. It enables students to lookup tutors and office hours for courses in the directory. It enables tutors to post office hours for students. It enables teachers to add and remove tutors for their courses. Database administrators can add courses and teachers. It was developed for a final project in the CS3200: Database Design class at Northeastern University during the Fall 2016 semester. Feel free to view and contribute to the source code for this project at the GitHub repository below.
